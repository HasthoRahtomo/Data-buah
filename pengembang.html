Website ini dibuat sebagai projek kecil saya dalam membuat sebuah web yang dapat berfungsi normal dengan hanya menggunakan ChatGPT. Tugas saya hanya memasukkan pertanyaan, melakukan copy-paste, mengubah file path, dan melakukan perubahan minor (misalnya membuat kerangka table). ChatGPT yang membuat kerangka, desain, dan memberikan informasi. Saya berumur 20 tahun saat website ini dibuat.

Saya membuat projek ini karena saya ingin tahu sejauh mana kemampuan chatGPT dalam mengembangkan web HTML berdasarkan pernyataan dan pertanyaan yang diberikan oleh user.
